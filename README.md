# Odin Recipes

This project builds a simple website for recipes. 

# Skills Applied

- build the initial structure with an index.html file with links to recipes
- create a directory that contains several recipes, each that link back to the index page
- embed an image, an image description heading, a paragraph describing the ingredients needed for the recipe, an unordered list of ingredients, and an ordered list with steps
- publish and view the site on the web
